# Sistema de Gerenciamento de Academia

## Descrição do Projeto

Estou redesenvolvendo esse projeto com o objetoivo de retestar e reforçar habilidades básicas de Programação Orientada a Objetos (POO) em Java (disciplina que tive no 2º semestre) com o conhecimento que adquiri até o 3º (redesenvolvimento). 

A aplicação utiliza Java Swing para a interface gráfica e foi criada como uma forma de praticar novamente conceitos fundamentais da disciplina, especialmente:
- classes e objetos;
- herança;
- polimorfismo;
- encapsulamento;
- coleções;
- programação orientada a eventos;
- organização de projeto em múltiplas classes.

O sistema tem como tema o gerenciamento de alunos de uma academia, permitindo cadastrar, visualizar e analisar dados dos alunos através de funcionalidades estatísticas.
